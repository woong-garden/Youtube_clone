## ****✨ 프로젝트 개요****

- 본 프로젝트는 YouTube API를 활용하여 유튜브 클론 코딩 웹앱을 구현하는 것을 목적으로 합니다.
- React를 기반으로 프로젝트를 구현하며, YouTube API를 사용하여 동영상 검색, 재생, 최신 인기동영상 기능을 구현합니다.

<br>
